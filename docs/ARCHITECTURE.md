# RuleWhisper — 架构设计讨论

> 状态：讨论中 | 起始：2026-07-12

---

## 1. 总体架构（已确定）

```
QQ消息 ──→ QQ Bot ──→ CLI ──→ 内置命令 → 直接返回给 QQ Bot
                          │
                          └──→ 对话命令(!前缀) → Agent
                                                     ├─ 调用 CLI 工具
                                                     └─ 组装回复 → QQ Bot → QQ
```

| 组件 | 职责 | 不可做的事 |
|------|------|-----------|
| QQ Bot | QQ消息 ↔ CLI 的透传适配层 | 不做任何逻辑判断 |
| CLI | 消息分流 + 内置命令执行 | 不直接访问 QQ |
| Agent | 自然语言理解、调用工具、生成回复 | 不直接访问 QQ |
