  1. 守秘人在掷骰前决定该线索为显明或隐秘
  2. 显明线索：玩家不需掷骰即可发现，对情节驱动有用(错失会中断进程时用)
  3. 隐秘线索：玩家需洞察检定，并且无论是否发现都不影响游戏有趣展开

侦查、聆听、心理学
  1. 守秘人设定难度并告知玩家
  2. 每位有机会的调查员进行洞察检定
  3. 通过且成功等级最高者获得线索；平局→交给所有通过者或只给技能最高者
  4. 失败：玩家知道漏了线索，可选 忽略(在调查员所知内行动) / 孤注一掷(更积极使用洞察) / 调查(猜测线索并围绕其调查)

  2. 房规集中在1-少数几项技能/属性，定义不同成功度含义
  3. 若允许某玩家重骰/交换属性→必须告知所有玩家并允许他人同样操作
